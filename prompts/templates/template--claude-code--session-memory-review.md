# Claude Code Session-To-Memory Review

Use this after a productive session to decide what belongs in durable memory,
what belongs in a reusable skill, and what should stay out.

## Template

```text
Review this session and propose updates to CLAUDE.md or a new skill only if the
instruction would be reused across future tasks.

Separate:
1. durable project facts
2. repeatable multi-step procedures
3. one-off details that should not be stored

Keep the proposal concise and explain why each item belongs in memory, a skill,
or neither.
```
