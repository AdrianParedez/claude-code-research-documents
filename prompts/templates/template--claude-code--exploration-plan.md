# Claude Code Exploration Plan

Use this when the task is large, ambiguous, or risky enough that you want a
plan before edits.

## Template

```text
I need to {{objective}}.

Before making any edits:
- read {{project_instructions}}
- inspect {{relevant_files_or_artifacts}}
- identify the files, services, tests, and risks that matter

Deliver:
1. the files you expect to touch
2. the implementation plan
3. open questions or ambiguities
4. a verification strategy

If requirements are ambiguous, ask targeted questions before proposing final
implementation details.
```
